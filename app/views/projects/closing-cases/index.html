Changing appeal types and closing cases

Sometimes appeals are closed. This can be because an appellant chooses to withdraw their appeal or because it changes type.

This area of the prototype shows all scenarios where a case is closed without a decision. It also shows the case details page after a decision has been made, but does not include the issuing a decision journey.

This prototype iteration covers withdrawing an appeal and changing an appeal type. If the appeal type is changed the usual assumption is that the appellant is asked to resubmit, if they are not then it is manually transferred to Horizon and the horizon ID linked.

Page examples

Static examples of the case details page for all of the different status on this ticket:

  • Case complete (displays dart letter)
  • Case withdrawn (appellant withraws appeal)
  • Case withdrawn (uploading) (appellant withraws appeal, file not virus checked yet)
  • Case withdrawn (virus check failed) (appellant withraws appeal, virus check failed)
  • Case closed (appeal type changed, appellant asked to resubmit)
  • Case awaiting transfer (appeal type changed, awaiting horizon ID to link)
  • Case transferred (appeal type changed, moved to horizon)
