Request missing evidence

Planning Inspectorate need case officers to keep additional documentation (evidence) separate. This is because inspectors might decide to ignore documents sent after a certain deadline closed. However it's important to list the additional evidence in the appropriate case documentation location.

This prototype journey is built to show the ability to add additional evidence to the Appellant case only. LPAQ is not represented here.
