Appeal a planning decision

Linked appeals

As a BO user I want to link / unlink appeals so that I can process cases in combination to align the decision process and economise CO/inspector time resource

Jira Ticket: BOATD-25

Version 2 {%- from "govuk/components/tag/macro.njk" import govukTag -%} {{govukTag({ text: "In progress", classes: "govuk-!-margin-left-2 govuk-tag--yellow" })}}

This version is redesigned based on allowing us to surface appeals stored in Horizon within the Back Office interface.

Features:

  • ability to link claims
  • ability to set claims as a parent or child
  • ability to unlink claims

Assumptions:

  • That we can display the data in the search table for claims in both back office and horizon.
  • That if the case officer needs more information than in the search table they will click to view the appeal or manually check horizon.
  • That not linking the appeal id is sufficient to indicate that the appeal is in horizon rather than back office.

What this doesn't do yet:

  • handle multiple linked appeals

What this doesn't do yet:

  • handle multiple linked appeals
